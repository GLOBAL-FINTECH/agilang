# AGILANG v1.3 Cross-Platform Native Runtime

AGILANG v1.3 keeps the Linux precompiled runtime from v1.2 and adds the release structure required to ship real native runtime artifacts for Windows and macOS.

## Supported release targets

| Platform tag | Shared library | Build host | Status |
|---|---|---|---|
| `linux-x86_64` | `libagilang_net_runtime.so` | `ubuntu-latest` | bundled when built on Linux |
| `windows-x86_64` | `agilang_net_runtime.dll` | `windows-latest` | generated by release workflow |
| `macos-x86_64` | `libagilang_net_runtime.dylib` | `macos-13` | generated by release workflow |
| `macos-arm64` | `libagilang_net_runtime.dylib` | `macos-latest` | generated by release workflow |

The package does not fake binaries for platforms the current build host cannot compile. Instead, `.github/workflows/release.yml` builds each runtime on its real operating system and merges the artifacts into the final release package.

## Runtime loading order

AGILANG resolves native runtime libraries in this order:

1. A developer-supplied library path.
2. A cached installed runtime under the AGILANG native runtime cache.
3. A bundled platform prebuilt runtime under `agilang/native/prebuilt/<platform-tag>/`.
4. Local compilation from `agilang/native/agilang_net_runtime.c`.

## Commands

```bash
agi runtime platform-matrix
agi runtime prebuilt-status
agi runtime install-prebuilt
agi runtime doctor
```

`platform-matrix` shows all supported release targets, the expected library name, the CI runner, and whether a shared library is physically bundled in the current package.

## Manual build commands

Linux:

```bash
python scripts/build_prebuilt_runtime.py --copy-root-native
```

Windows PowerShell:

```powershell
python scripts\build_prebuilt_runtime.py --copy-root-native
```

macOS:

```bash
python scripts/build_prebuilt_runtime.py --copy-root-native
```

## Security verification

Every prebuilt directory contains a `manifest.json` with SHA-256 hashes for the generated artifacts. Deployment systems should compare the manifest hash against the installed library before loading it in high-security environments.
