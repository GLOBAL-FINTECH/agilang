What is AGILANG?

AGILANG is a modular programming language and runtime for building modern applications. It is not only a blockchain project. Blockchain is one application domain inside a broader language and app development platform.

.agi source files

Write backend logic, routes, services, CLI scripts, and runtime modules.

.ags templates

Create reactive pages with fetch/live bindings and clean layout composition.

One CLI

Generate apps, run programs, serve web apps, test projects, and create blockchain starters.

Quick start

git clone https://github.com/GLOBAL-FINTECH/agilang.git
cd agilang
python -m pip install -e .
agi --version
agi run hello.agi

Hello world

fn main() -> i32:
    print("Hello from AGILANG")
    return 0

Application starter kits

Web app

agi new my-web-app
cd my-web-app
agi serve src/main.agi --host 127.0.0.1 --port 8000

CMS/blog starter

git checkout blog
agi serve src/main.agi --host 127.0.0.1 --port 8000

Blockchain generator

AGILANG can generate a complete configurable blockchain starter, including chain runtime files, Beacon configuration, staking configuration, RPC configuration, Ethereum private-fork configuration, wallet examples, and runbook documentation.

agi new my-chain --template blockchain
cd my-chain
agi run
agi run src/chain.agi
agi run src/beacon.agi

Production real-value networks require independent security review, hardened networking, validator key isolation, DoS protection, monitoring, and economic review.
