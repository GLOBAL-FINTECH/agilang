# AGILANG native runtime: windows-x86_64

This directory is a release target for AGILANG precompiled native runtime artifacts.

Real binaries for this platform are generated by .github/workflows/release.yml on the matching OS runner and copied here during release packaging. The current Linux build environment cannot safely produce or verify this platform's native shared library, so this package does not include a fake binary.

Expected files after the release workflow runs:

- shared library for this platform
- static archive/object where supported
- manifest.json with SHA-256 hashes
